# Integrals

The `<Plot/>` module can render areas under curves by initializing the `FunctionDatum`'s `integrate` property. For example, say we wanted to the area beneath the curve <Maths val={`x^3`} block={false}/> from -1 to 5. That is:

<Maths val={`\\int_{-1}^{5} x^3`} />

we can do so by writing:

```javascript
<Plot
  data={[{ f: (x) => x ** 3, integrate: [-3, 5, 'x'] }]}
  domain={[-10, 10]}
  range={[-10, 10]}
/>
```

which renders:

<Plot
  data={[{ f: (x) => x ** 3, integrate: [-3, 5, 'x'] }]}
  domain={[-10, 10]}
  range={[-10, 10]}
/>
