# Annotating Plots

The graph rendered by the `<Plot/>` module can be annotated by passing a `TextDatum` object. This object has the form:

```bash
{t: <string>, x: <number>, y: <number>, w: <number>, h: <number>}
```

The fields are defined as such:

- `t` is the text-content of the label.
- `x` is the ùë•-coordinate.
- `y` is the ùë¶-coordinate.
- `w` is the width of the textbox.
- `h` is the height of the textbox.

For example, say we have the following plot:

<Plot data={[{ f: (x) => 1/x }]} />

To render the function's definition, we can write:

```jsx
<Plot data={[{ f: (x) => 1/x }, { t: 'f(x)=\\dfrac{1}{x}', x: 3, y: 3 }]} />
```

The result:

<Plot data={[{ f: (x) => 1/x }, { t: 'f(x)=\\dfrac{1}{x}', x: 3, y: 3 }]} />
