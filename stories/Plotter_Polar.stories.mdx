# Polar Plots

For some functions, Cartesian coordinates don't yield very interesting plots. For example, consider the function <TeX math="f(x) = \sin(2x)\cos(2x)"/>. Plotting this function with Cartesian coordinates, we get:

<Plot data={[{ f: x => Math.sin(2 * x) * Math.cos(2 * x) }]} />

This doesn't look all that interesting. We can change that by plotting the function with polar coordinates:
