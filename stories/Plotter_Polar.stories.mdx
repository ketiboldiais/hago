# Polar Plots

For some functions, Cartesian coordinates don't yield very interesting plots. For example, consider the function:

<Maths val={`f(t) = \\sin(2t)\\cos(2t)`} />

Plotting this function with just the `<Plot/>` module, we write:

```javascript
<Plot data={[{ f: (x) => Math.sin(2 * x) * Math.cos(2 * x) }]} />
```

giving us the plot:

<Plot data={[{ f: (x) => Math.sin(2 * x) * Math.cos(2 * x) }]} />

This doesn't look all that interesting. We can change that by plotting the function using the `<Polar/>` component instead. Writing:

```javascript
<Polar data={[{ f: (t) => Math.sin(2 * t) * Math.cos(2 * t) }]} />
```

we get:

<Polar data={[{ f: (t) => Math.sin(2 * t) * Math.cos(2 * t) }]} />
