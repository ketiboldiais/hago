# Function Plotters

Hago's `Plot` module provides a way to plot functions using native JavaScript functions. For example, writing:

```jsx
<Plot data={[{ f: x => Math.sin(x) }]} />
```

Results in:

<Plot data={[{ f: x => Math.sin(x) }]} />

Read further for additional examples.
