# Plot 3D

For more interactive parametric functions, we can use the `Plot3D` module. For example, the function:

<Maths val={`z(x,y) = \\sin \\left( \\sqrt{x^2 + y^2} \\right)`} />

can be rendered by writing:

```jsx
<Plot3D z={(x, y) => Math.sin(Math.sqrt(x ** 2 + y ** 2))} />
```

to render:

<Plot3D z={(x, y) => Math.sin(Math.sqrt(x ** 2 + y ** 2))} />
