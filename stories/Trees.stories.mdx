# Tree

The `<Tree/>` component will render a generic tree:

<Tree
  data={[
    ['a', ''],
    ['b', 'a'],
    ['c', 'a'],
    ['e', 'c'],
    ['f', 'c'],
    ['g', 'b'],
    ['h', 'b'],
    ['i', 'g'],
    ['j', 'g'],
    ['k', 'e'],
    ['l', 'e'],
    ['m', 'h'],
    ['n', 'h'],
  ]}
  slim={10}
  id="Tree1"
/>

## Binary Trees

If a tuple is passed to the nested array of nodes, a binary tree is assumed and rendered. For example, the code:

```javascript
<Tree
  data={[
    [0, [2, 3]],
    [2, [17, 19]],
    [3, [36, 7]],
    [17, [25, null]],
  ]}
/>
```

renders the tree:

<Tree
  data={[
    [0, [2, 3]],
    [2, [17, 19]],
    [3, [36, 7]],
    [17, [25, null]],
  ]}
/>
