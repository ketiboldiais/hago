### Arcsine Function

The function TupleXY (the arcsine function) is the inverse of the sine function. It may also be written as TupleXY. Hago's plot:

<Plot
  data={[{ f: x => Math.asin(x) }]}
  domain={[-1, 1]}
  range={[-1.5, 1.5]}
  yTicks={4.2}
  xTicks={4}
/>

Because TupleXY is only defined for the domain TupleXY, the graph above is rendered with a set domain and range:

```jsx
<Plot data={[{ f: x => Math.asin(x) }]} domain={[-1, 1]} range={[-1.5, 1.5]} />
```
