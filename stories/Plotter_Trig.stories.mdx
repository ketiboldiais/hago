### Arcsine Function

The function <TeX math="y = \arcsin(x)"/> (the arcsine function) is the inverse of the sine function. It may also be written as <TeX math="y = \sin^{-1}(x)"/>. Hago's plot:

<Plot
  data={[{ f: x => Math.asin(x) }]}
  domain={[-1, 1]}
  range={[-1.5, 1.5]}
  yTicks={4.2}
  xTicks={4}
/>

Because <TeX math="y = \arcsin(x)"/> is only defined for the domain <TeX math="\{ x \in \R : -1 \leq 0 \leq 1 \}"/>, the graph above is rendered with a set domain and range:

```jsx
<Plot data={[{ f: x => Math.asin(x) }]} domain={[-1, 1]} range={[-1.5, 1.5]} />
```
