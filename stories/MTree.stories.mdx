# M-way Trees

    { c: 'B', p: 'A', v: [2, 7] },
    { c: 'C', p: 'A', v: [4, 1] },
    { c: 'E', p: 'B', v: [9, 5] },
    { c: 'N', p: 'B', v: [9, 5] },
