# Cumulative Relative Frequency

Passing the option `["cumulative relative frequency", 𝑘]` where 𝑘 is a header will generate a column of the cumulative frequencies. E.g.,

```javascript
<DataTable
  data={[5, 6, 3, 3, 2, 4, 7, 5, 2, 3, 5, 6, 5, 4, 4, 3, 5, 2, 5, 3]}
  output={[['cumulative relative frequency', 'y']]}
/>
```

renders:
