# Max

To return the max of the 𝑦-values, we write:

```javascript
<DataTable
  data={[5, 6, 3, 3, 2, 4, 7, 5, 2, 3, 5, 6, 5, 4, 4, 3, 5, 2, 5, 3]}
  output={[['frequency', 'y']]}
  include={['mean', 'max']}
/>
```

rendering:
