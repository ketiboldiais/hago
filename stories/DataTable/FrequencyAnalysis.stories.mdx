# Frequency Analysis

A frequency analysis can be conducted by passing the `frequency` to the output.

```javascript
<DataTable
  data={[5, 6, 3, 3, 2, 4, 7, 5, 2, 3, 5, 6, 5, 4, 4, 3, 5, 2, 5, 3]}
  output={[['frequency', 'y']]}
/>
```

generates a frequency table for the `"Datum"` values:

<DataTable
  data={[5, 6, 3, 3, 2, 4, 7, 5, 2, 3, 5, 6, 5, 4, 4, 3, 5, 2, 5, 3]}
