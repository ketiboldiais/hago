# Secant Line

The `FunctionDatum` object contains a field called `secant`. This field takes a `SecantDatum` object:

  data={[
    {
      f: (x) => x ** 2 - 2 * x + 3,

        x0: 1,
        x1: 5,
      },
    },
  ]}
  domain={[-1, 6]}
  range={[-5, 35]}
/>
