# Multivariable Functions

Although most individuals are familiar with single-variable functions — the famous — functions can take any number of variables. They're just a special kind of relation after all. For example, consider the function:

<Maths val={"f(x) = \\sin\\left(\\dfrac{\\sqrt{x^2+y^2}}{5}\\right)\\pi"}/>

To plot these functions, we can use the `Multiplot` module. For example, the code:

```jsx
<Multiplot
  id={'multiplot_example_1'}
  data={[
    {
      f: (x, y) => Math.sin((Math.sqrt(x * x + y * y) / 5) * Math.PI),
      scale: 70,
    },
  ]}
/>
```
