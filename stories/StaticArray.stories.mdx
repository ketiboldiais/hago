# Static Arrays

Static arrays are a simple data structure to render. Hago renders them with the `StaticArray` component:

<StaticArray data={[1, 2, 3, 4, 5, 6, 7]} />

## Annotations

Annotations can be appended at the top of elements.

<StaticArray data={[1, 2, 3, 4, { val: 5, ant: 'p' }, 6, 7]} />

## Pointers

Pointers can be appended at the bottom elements.

<StaticArray data={[1, { val: 2, ptr: 'C' }, 3, 4, 5, 6, 7]} />

## Pointers & Annotations

Pointers and annotations can be appended together:

<StaticArray data={[1, 2, 3, 4, { val: 5, ant: 'p', ptr: 'C' }, 6, 7]} />

## Resizing

Note that long arrays increase the container's height in order to scale horizontally. This results in additional bottom padding in the container:

<StaticArray data={[5, 6, 1, 2, 3, 4, 5, 4, 3, 2, 0, 1, 2, 3, -2, 4]} />

Depending on your site's layout, you may see this pushing the content just below the diagram towards the bottom like this. To fix this, set the `cheight` (container height) property to some value between `0` and `1`:

<StaticArray
  data={[5, 6, 1, 2, 3, 4, 5, 4, 3, 2, 0, 1, 2, 3, -2, 4]}
  cheight={0.2}
/>

The rendering code:

```jsx
<StaticArray
  data={[5, 6, 1, 2, 3, 4, 5, 4, 3, 2, 0, 1, 2, 3, -2, 4]}
  cheight={0.2}
/>
```
